# 视频转提示词网站 · 开发周期闯关文档

**技术栈：** Vue 3 + TypeScript + Tailwind CSS / Node.js + FastAPI（AI 分析层）  
**版本：** v1.0  
**日期：** 2026-04-12

> 本文档将开发周期拆解为 **4 大关卡 / 16 周**，每关包含若干「子关」，每个子关有明确的验收标准（DoD）。完成所有验收项才可进入下一关。

---

## 全局技术栈说明

| 层级 | 技术选型 |
|------|---------|
| 前端框架 | Vue 3 + TypeScript + Vite |
| 前端样式 | Tailwind CSS + shadcn-vue |
| 状态管理 | Pinia |
| 路由 | Vue Router 4 |
| HTTP 客户端 | Axios + VueQuery |
| 后端框架 | Node.js + Express / Fastify |
| 数据库 | PostgreSQL + Prisma ORM |
| 缓存/队列 | Redis + BullMQ |
| 视频处理 | FFmpeg（子进程调用） |
| AI 分析 | 智谱 GLM-4.1V-Thinking-FlashX API |
| 存储 | 阿里云 OSS / AWS S3 |
| 部署 | Docker + Docker Compose + Nginx |
| 测试 | Vitest（前端）+ Jest（后端）|
| CI/CD | GitHub Actions |

**数据库 Schema**

```prisma
model Analysis {
  id          String       @id @default(cuid())
  userId      String
  fileId      String
  status      AnalysisStatus
  config      Json         // 采样密度、平台、语言
  frames      Frame[]
  overallPrompt Json?      // 整体提示词（各平台）
  styleTags   String[]
  createdAt   DateTime     @default(now())
  user        User         @relation(fields: [userId], references: [id])
}

model Frame {
  id          String   @id @default(cuid())
  analysisId  String
  timestamp   Float    // 时间戳（秒）
  thumbUrl    String   // 缩略图 URL
  rawAnalysis Json     // AI 原始分析结果
  prompts     Json     // 各平台提示词
  analysis    Analysis @relation(fields: [analysisId], references: [id])
}

enum AnalysisStatus {
  PENDING
  EXTRACTING
  ANALYZING
  DONE
  FAILED
}
```

## 附录：风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| AI API 调用成本超预期 | 高 | 高 | 设置每日 API 费用上限告警；对相同视频缓存结果 |
| FFmpeg 视频格式兼容性问题 | 中 | 中 | 提前测试主流格式；提供格式转换降级方案 |
| 视频链接平台反爬 | 高 | 中 | 降级提示用户本地下载上传；保持解析库更新 |
| 高并发时队列积压 | 中 | 高 | BullMQ 动态扩容 Worker；设置用户排队提示 |
| 版权纠纷 | 低 | 高 | 完善版权声明；不存储原始视频；法律顾问审核条款 |
